üìñ Educator Facilitation Guide

  • Opening Script: "Today, we're going to use a special toolkit to help us talk about our feelings and how we work together. Let's have fun and support each other!"
  • Transition Script: "Now that we've finished this activity, let's take a moment to reflect on what we learned before we move on."
  • Closure Script: "Thank you for sharing your thoughts. Remember, these skills help us in and out of the classroom!"
  • Troubleshooting: If students are shy, allow think-pair-share or journaling before group sharing. For off-task behavior, use the timer and clear expectations.
  • Age Adaptations: For younger students, use simpler language and shorter activities. For older students, encourage deeper discussion and self-reflection.

üöÄ Quick Start for Educators

  1. Choose an Activity: Select a dice or scenario card set from the dashboard.
  2. Prep Materials: Print or project the resource. Assemble dice if needed (5-10 min).
  3. Set the Timer: Use the built-in timer for structured discussions (2-5 min recommended).
  4. Facilitate: Read the prompt aloud, use guiding questions, and encourage participation.
  5. Reflect: Use the reflection tools or closing script to wrap up.
